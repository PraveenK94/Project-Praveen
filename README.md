# Database Management Projects Collection

## Overview
Collection of MySQL and PostgreSQL database projects demonstrating various aspects of database management and analysis.

## Projects Included
1. Academic Management System
2. Student Database Management System
3. Event Management System
4. OLAP Operations

## Technical Requirements
- PostgreSQL 12+
- Mysql command-line tool
- Basic SQL knowledge

## Installation
1. Install My SQL & PostgreSQL
2. Clone repository
3. Run setup scripts

## Usage
Each project contains:
- Table creation scripts
- Sample data insertion
- Query examples
- Analysis functions

## Common Commands
```psql
-- Create Database
CREATE DATABASE database_name;

-- Connect to Database
\c database_name

-- Create Table
CREATE TABLE table_name (...);

-- Insert Data

-- Basic Queries
SELECT * FROM table_name;
